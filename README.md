# Temporal Portfolio: A Multi-Era Developer Journey

A highly interactive, cinematic web portfolio that takes users through a journey across different "eras" of development. Built with React, Vite, and advanced animation libraries.

## 🚀 Overview

This portfolio is designed as a temporal journey, starting from a terminal-based boot sequence and progressing through different stages of digital evolution. It uses a combination of immersive storytelling and modern web technologies to showcase technical skills and projects.

## 🛠️ Tech Stack

- **Framework**: [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: 
  - [Framer Motion](https://www.framer.com/motion/) (for UI interactions and transitions)
  - [GSAP](https://gsap.com/) (for scroll-triggered effects)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## 🕰️ Timeline Eras

### 1. The Origin (Era 1)
Focuses on the foundations—HTML, CSS, and JavaScript. Features a retro terminal interface that simulates the early learning process.

### 2. The Builder (Era 2)
Represents the era of frameworks and scalable architecture. Showcases core technical proficiencies in a modern, structured layout.

### 3. The Architect (Era 3)
Highlights complex, intelligent systems. Features interactive project cards and data flow visualizations.

### 4. The Future (Era 4)
A forward-looking section dedicated to emerging technologies like Agentic AI and Web3, featuring holographic UI elements and autonomous agent simulations.

## 🖱️ Interactive Features

- **Custom Cursor**: Context-aware cursor that changes its appearance based on the current era.
- **Timeline Navigation**: A sticky side navigation that tracks progress and allows jumping between eras.
- **Temporal Interface**: A terminal-based "boot sequence" that welcomes users to the experience.
- **Scanline Effects**: Subtle CRT-style scanlines used in earlier eras to enhance the aesthetic.

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Navigate to the `app` directory:
   ```bash
   cd app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Create a production build:
```bash
npm run build
```
